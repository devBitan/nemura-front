# nemura-front

## Tecnologies
- Framework: vue3
- Language: javascript
- Http service: axios
- Persistence: pinia
- Routes: vue-router

## Start the project

clone the porject
- git clone https://github.com/devBitan/nemura-front.git
check the route
  -yourpc/nemura-front/nemura-front
install dependencies
  - npm i or npm install

## Run the project
npm run dev
